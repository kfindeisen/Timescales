void kpftimes::acWindow ( const DoubleVec &  times,
const DoubleVec &  offsets,
DoubleVec &  wf 
)

Calculates the autocorrelation window function for a time sampling.

The window function is defined in [3].

Parameters
[in]timesTimes at which data were taken
[in]offsetsThe time grid over which the autocorrelation function should be calculated.
[out]wfThe value of the window function at each offset.
Precondition
times contains at least two unique values
times is sorted in ascending order
offsets contains at least two unique elements
offsets contains only nonnegative values
offsets is uniformly sampled from 0 to some maximum value. This requirement will be relaxed in future versions.
Postcondition
wf.size = offsets.size()
wf[i] is the Scargle autocorrelation function for a constant function evaluated at offsets[i], for all i
Performance
O(FN) time, where N = times.size() and F = offsets.size()
Exceptions
kpftimes::except::BadLightCurveThrown if times has at most one distinct value.
kpfutils::except::NotSortedThrown if times is not in ascending order.
kpftimes::except::NegativeFreqThrown if some offsets are negative.
std::invalid_argumentThrown if offsets has at most one distinct value, or if it is not uniformly sampled.
std::bad_allocThrown if there is not enough memory to perform the calculations.
Exception Guarantee
The function arguments are unchanged in the event of an exception.

Calculates the autocorrelation function for a time series.

Parameters

[in]maxFreqThe maximum frequency to consider when calculating the autocorrelation function.
[out]acfThe value of the autocorrelation function at each offset.
Note
Increasing maxFreq will increase the time resolution of acf at the cost of making the entire function noisier.
Precondition
times contains at least two unique values
times is sorted in ascending order
fluxes.size() = times.size()
fluxes[i] is the flux of the source at times[i], for all i
offsets contains at least two unique elements
offsets contains only nonnegative values
offsets is uniformly sampled from 0 to some maximum value. This requirement will be relaxed in future versions.
maxFreq is positive
Postcondition
acf.size() = offsets.size()
acf[i] is the Scargle autocorrelation function evaluated at offsets[i], for all i
Performance
O(FN) time, where N = times.size() and F = offsets.size()
Exceptions
kpftimes::except::BadLightCurveThrown if times has at most one distinct value.
kpfutils::except::NotSortedThrown if times is not in ascending order.

The function arguments are unchanged in the event of an exception.
