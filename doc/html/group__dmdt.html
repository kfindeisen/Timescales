Calculates all Delta-m Delta-T pairs.

deltaT is returned sorted in ascending order. Pairs of observations with the same time separation may be returned in any order.

Parameters:
[in]timesTimes at which data were taken
[in]fluxesFlux measurements of a source
[out]deltaTA list of the time intervals between all pairs of sources.
[out]deltaMA list of the magnitude difference between each pair in deltaT.
Precondition:
times contains at least two unique values
times is sorted in ascending order
fluxes is of the same length as times
fluxes[i] is the flux of the source at times[i], for all i
Postcondition:
if N == length(times), length(deltaT) == length(deltaM) == N(N-1)/2
Exceptions:
invalid_argumentThrown if the preconditions on times or length(fluxes) are violated.
Performance:
O(times.size()^2) time
