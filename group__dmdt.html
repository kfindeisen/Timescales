Detailed Description

Support for ΔmΔt plots.

Brute-force implementation of ΔmΔt plots. Also provides functions for basic ΔmΔt summary statistics.

Computes the quantile of pairs of magnitudes found in each Δt bin of a ΔmΔt plot.

Parameters
[in]deltaTThe Δt values of the ΔmΔt plot.
[in]deltaMThe corresponding Δm values of the ΔmΔt plot.
[in]binEdgesA vector containing the (N+1) boundaries of the N Δt bins in which to calculate quantiles.
[out]quantsA vector containing the quantiles within each bin.
[in]qThe quantile to calculate.
Precondition
deltaT.size() = deltaM.size()
deltaT is sorted in ascending order
binEdges is sorted in ascending order
0 < q < 1
deltaT does not contain any NaNs
deltaM does not contain any NaNs
binEdges does not contain any NaNs
Postcondition
quants.size() = binEdges.size() - 1
For all i ∈ [0, binEdges.size()-1], quants[i] contains the qth quantile of deltaM, given deltaT ∈ [binEdges[i], binEdges[i+1])
Performance
O(N log N) time, where N = deltaT.size()
o(N log N - N log M) time, where N = deltaT.size() and M = binEdges.size()
Exceptions
std::bad_allocThrown if there is not enough memory to store the bin fractions.
std::invalid_argumentThrown if q is not in (0, 1) or if deltaT.size() ≠ deltaM.size()
kpfutils::except::NotSortedThrown if either deltaT or binEdges is unsorted.
Exception Guarantee
The function arguments are unchanged in the event of an exception.
void kpftimes::dmdt ( const DoubleVec &  times,
const DoubleVec &  mags,
DoubleVec &  deltaT,
DoubleVec &  deltaM 
)

Calculates a ΔmΔt plot.

Parameters
[in]timesTimes at which mags were taken
[in]magsMagnitude measurements of a source
[out]deltaTA list of the time intervals between all pairs of sources.
[out]deltaMA list of the magnitude difference between each pair in deltaT.
Precondition
times contains at least two unique values
times is sorted in ascending order
mags.size() = times.size()
mags[i] is the magnitude of the source at times[i], for all i
Postcondition
deltaT.size() = deltaM.size() = N(N-1)/2, where N = times.size()
deltaT is sorted in ascending order
Each element of deltaT represents the absolute time separation of a unique pair of values in times, and each element of deltaM represents the absolute magnitude separation of a corresponding pair of values in mags
Performance
O(N2) time, where N = times.size()
Exceptions
kpftimes::except::BadLightCurveThrown if times has at most one distinct value.
kpfutils::except::NotSortedThrown if times is not in ascending order.
std::invalid_argumentThrown if times and mags have different lengths.
std::bad_allocThrown if there is not enough memory to compute the ΔmΔt plot
Exception Guarantee
The function arguments are unchanged in the event of an exception.

Computes the fraction of pairs of magnitudes above some threshold found in each Δt bin of a ΔmΔt plot.

Parameters
[in]deltaTThe Δt values of the ΔmΔt plot.
[in]deltaMThe corresponding Δm values of the ΔmΔt plot.
[in]binEdgesA vector containing the (N+1) boundaries of the N Δt bins in which to count high-Δm pairs.
[out]fracsA vector containing the fraction of Δm values in each bin that exceed threshold.
[in]thresholdThe characteristic magnitude difference, in magnitudes, above which Δm values are to be counted.
Precondition
deltaT.size() = deltaM.size()
deltaT is sorted in ascending order
binEdges is sorted in ascending order
deltaT does not contain any NaNs
deltaM does not contain any NaNs
binEdges does not contain any NaNs
Postcondition
fracs.size() = binEdges.size() - 1
For all i ∈ [0, binEdges.size()-1], fracs[i] contains the fraction of deltaM > threshold, given deltaT ∈ [binEdges[i], binEdges[i+1]).
Performance
O(M + N) time, where N = deltaM.size() and M = binEdges.size()
Exceptions
std::bad_allocThrown if there is not enough memory to store the bin fractions.
kpfutils::except::NotSortedThrown if either deltaT or binEdges is unsorted.
std::invalid_argumentThrown if deltaT.size() ≠ deltaM.size()
Exception Guarantee
The function arguments are unchanged in the event of an exception.
